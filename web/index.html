fast, reliable wildcard dns

When a *.netip.cc domain is resolved, the embedded IP address is returned as the corresponding A record:

                        10.20.30.40.netip.cc
             api.system.10.20.30.40.netip.cc
a.b.c.d.e.f.g.h.i.j.k.l.10.20.30.40.netip.cc
           10.20.30.40.netip.cc
   api.sys.10.20.30.40.netip.cc
 a.b.c.d.e.10.20.30.40.netip.cc

To use it, just make up a name! When you need a DNS entry, use name.$IP.netip.cc. You can also run your own netip resolver, on your own servers, using your own base domain.

You can download the netip resolver:

  • For [L]inux 64-bit
  • For [M]ac OS X 64-bit
  • or, just get the [S]ource Code

Usage is straightforward:

$ ./netip -n your.base.tld -b ip.ad.dr.ess:53 &
$ ./netip -n your.base.tld \
          -b ip.ad.dr.ess:53 &
